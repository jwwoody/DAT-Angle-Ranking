# DAT-Angle-Practice
Randomly generated angle ranking practice problems for the DAT. User specified tolerances.

## Setup

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd DAT-Angle-Ranking
   ```

2. **Create virtual environment**
   ```bash
   python -m venv dat
   source dat/bin/activate  # On Windows: dat\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment setup**
   ```bash
   cp .env.example .env
   # Edit .env file and set a secure SECRET_KEY
   ```

5. **Run the application**
   ```bash
   python app.py
   ```

6. **Open in browser**
   Go to http://localhost:5000
